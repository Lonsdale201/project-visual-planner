# Blueprints

These files are ready-to-import project templates for ProjectPlanner.

## Import
1. Open the app.
2. Click **Import**.
3. Select a `.json` blueprint file from this folder.

## Available
- `project-manager-saas.knitflow.json` - AI-ready Project Manager SaaS architecture with milestones, services, integrations, DB schema, and code nodes.
- `wp-lms-plugin.knitflow.json` - WordPress LMS plugin architecture with custom table CRUD, ORM/DTO/Presenter patterns, and split domain flows.
